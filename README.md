# Proyecto_APT_SeniorFitness
Aplicacion senior Fitness: app para adultos mayores con recomendaciones personalizadas e IA

---

SeniorFitness App — Entrenadora Virtual para Adultos Mayores

SeniorFitness es una aplicación móvil desarrollada con Flutter que busca apoyar a adultos mayores en la realización de actividad física mediante rutinas simples, seguimiento de progreso y mensajes motivacionales generados con inteligencia artificial.


---

1. Objetivo del Proyecto

Crear una aplicación accesible que incentive la actividad física en adultos mayores, ofreciendo ejercicios guiados, notificaciones motivacionales y un sistema de rachas que refuerce la constancia.


---

2. Tecnologías Utilizadas

Frontend: Flutter, Dart, Lottie, Google Fonts
Backend: Firebase Authentication, Firestore, Storage
IA: Gemini API (google_generative_ai)
Otros: Flutter Local Notifications

5. Instalación y Ejecución

Frontend:

cd frontend/senior_fitness_nuevo
flutter pub get
flutter run

Backend (Firebase): Configurar Firebase y ejecutar:

flutterfire configure

Generar APK:

flutter build apk --release
